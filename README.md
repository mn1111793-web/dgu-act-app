# dgu-act-app

Minimal Android Studio project scaffold recreated as a text-only repository.

## Included files

- Root Gradle settings and plugin configuration.
- Android application module build script.
- `AndroidManifest.xml`.
- Kotlin source files for a simple Jetpack Compose activity.
- `strings.xml` and theme definitions.

## Notes

- Binary assets, icons, generated files, and other non-text resources are intentionally omitted.
- Open the repository root in Android Studio; the IDE can regenerate local/project metadata as needed.
